# Preview Plugin for Geany

Provides a preview pane in the Geany sidebar to view the formatting of several lightweight markup languages as they are edited.

![screenshot](docs/screenshot-main-window.png)

## Supported formats

AsciiDoc, DocBook, Fountain, HTML, LaTeX, Markdown, MediaWiki, reStructuredText, Textile, and Txt2Tags

## Features

* The preview is updated as the document is edited
* HTML, Markdown, and Fountain are supported directly
* Other document formats can be processed with auxiliary programs
* Rendering of each format can be customized with stylesheets
* Dark and light themes are supported
